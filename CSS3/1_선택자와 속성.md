## 1_선택자와 속성

### 선택자

1. 전체 선택자 : * {}
   - HTML 페이지 내부의 태그 모두 선택
2. 태그 선택자 : 태그, 태그, 태그, 태그
   - 특정 태그를 모두 선택
3. 아이디 선택자 : #아이디
   - 특정 id 속성이 있는 태그 선택
   - id 속성은 웹 페이지 내부에서 중복되면 안된다.
4. 클래스 선택자 : .클래스
   - 특정 클래스가 있는 태그 선택
5. 속성 선택자 : 특정 속성과 값이 있는 태그를 선택할 때 사용
   - 선택자[속성 = 값]
   - input[type]
   - input[type = ["text"]
6. 후손 선택자
   - 선택자a 선택자b
   - 특정한
