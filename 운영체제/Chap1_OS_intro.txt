Chapter 1. 운영체제 Intro

운영체제의 역할? 
- System Resource 관리자 (시스템 자원 = 컴퓨터 하드웨어 = CPU, DRAM, RAM, I/O devices, 저장매체 등)

컴퓨터 하드웨어는 스스로 할 수 있는 것이 없다.
ex) 운영체제가 각 프로그램 별 CPU 사용량을 할당하는 것.

대표적 운영체제 : Windows, Mac, UNIX
UNIX 계열의 OS : 사용법이나 OS 구조가 유사. ex) LINUX(리눅스)

운영체제는 만들기 쉽지 않다.

운영체제의 역할2 : User와 Computer간의 communication 지원

운영체제의 역할3 : 컴퓨터 하드웨어와 프로그램을 제어
ex) 응용프로그램과 하드웨어 사이의 communication 지원

소프트웨어 = 운영체제, 응용 프로그램

운영체제는 응용 프로그램을 관리한다.
- 응용 프로그램을 실행시켜준다.
- 응용 프로그램간의 권한을 관리해준다.
- 응용 프로그램을 사용하는 사용자도 관리한다.
- 응용 프로그램에 문제가 있어서 다운이 된다. << 운영체제가 관리하는 것임

따라서 운영체제의 목표 : 사용자가 사용하는 응용 프로그램이 효율적으로 적절하게 동작하도록 지원한다.

운영체제는 응용 프로그램이 요청하는 시스템 자원을 효율적으로 분배하고, 지원하는 SW이다.

운영체제의 위치? : 저장매체(SSD/HDD)에 저장(설치) 됨
